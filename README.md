# 灵枢 Nodus — 统一智能体

Hermes（思维）+ NanoBot（通讯）+ OpenClaw（执行）= 单一 Python 进程。

## 安装

```powershell
irm https://raw.githubusercontent.com/yeyuecho/Nodus/main/scripts/install.ps1 | iex
```

等待安装完成，中间会提示你输入 `DEEPSEEK_API_KEY`。看到"安装完成"就成功了。

## 使用

```bash
nodus start     # 启动
nodus doctor    # 诊断
nodus config    # 配置
nodus status    # 状态
nodus version   # 版本
```
